01 / About Me

스타트업과 헬스케어 서비스 회사를 거치며, 앱·웹·관리자 사이트를 구축하고 개선해 왔습니다. 모노레포 구조를 경험해보며 공통 컴포넌트를 설계·정비해 재사용성을 높였고, 사용자의 시선으로 실제 사용 맥락을 고려해 UI/UX를 반복적으로 다듬어 왔습니다.

기존 11명이었던 개발팀이 2명으로 축소된 상황에서도 기획·디자인·운영 부서와 긴밀하게 소통하며 요구사항을 구체화하고, 우선순위를 조율해 프로젝트를 완수했습니다. AI를 코드 작성에만 사용하는 것이 아니라 요구사항 정리, 구현 방향 검토, 오류 원인 분석, 테스트 및 문서화에 활용해 개발 속도와 업무 효율을 높였습니다. 이를 통해 제한된 인력 환경에서도 서비스 품질을 유지하며 프론트엔드와 백엔드 업무를 유연하게 수행할 수 있는 역량을 쌓았습니다.

AI 기반 개발 워크플로를 구축해 프로젝트 설계부터 구현, API 연동, Git 작업까지 개발 전 과정을 효율화했습니다. 브레인스토밍 스킬로 요구사항과 구조 설계를 구체화하고, Figma MCP를 통해 디자인을 분석해 초기 UI 코드를 빠르게 생성했습니다. Swagger 명세를 바탕으로 API 응답 구조와 호출 흐름을 신속하게 반영했으며, Git 작업은 변경사항 분석과 커밋 메시지 추천, 승인 기반 커밋·푸시 과정으로 자동화했습니다. AGENTS.md에는 작업 규칙과 Git 정책, 수정 범위를 정의해 AI 에이전트의 자율성과 통제 가능성을 함께 확보했습니다. 그 결과 예상 개발 기간 1~2개월의 프로젝트를 약 2주 만에 완료했습니다.
