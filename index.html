01 / About Me

4년 차 프론트엔드 개발자로서 스타트업과 헬스케어 서비스 회사를 거치며, 앱·웹·관리자 사이트를 구축하고 개선해 왔습니다. 모노레포 구조 아래에서 공통 컴포넌트를 설계·정비해 재사용성을 높였고, 실제 사용 맥락을 고려해 UI/UX를 반복적으로 다듬어 왔습니다.

기존 11명이었던 회사에서 2명으로 인원수가 감소하는 상황에서도 기획·디자인·운영 등 타 부서와 소통하며 요구사항을 명확히 정리하고 합의를 이끌어 프로젝트를 완수한 경험이 있습니다.

개인 학습으로 Next.js 16 + React 19 기반의 AI 챗봇 프로젝트를 진행했습니다. Ollama 기반 로컬 LLM을 연동해 실시간 스트리밍 응답(ReadableStream)을 구현하고, Zustand persist와 Markdown 렌더링을 적용했습니다. 현재는 React Native로 어머니들의 육아 서포트 앱 '맘마(Mamma)'를 개발 중입니다.
